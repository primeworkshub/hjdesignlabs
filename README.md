# HT Design Labs Website

A professional, SEO-optimized single-page website for HT Design Labs - a global design agency specializing in web design, graphic design, and creative services.

## Features

### SEO Optimization
- **Meta Tags**: Comprehensive meta tags including title, description, keywords, and Open Graph tags
- **Schema.org Markup**: Structured data for better search engine understanding
- **Semantic HTML**: Proper use of HTML5 semantic elements (header, nav, section, footer)
- **Keyword-Rich Content**: Naturally integrated keywords throughout the content
- **Mobile-First Design**: Responsive design optimized for all devices
- **Fast Loading**: Optimized CSS and JavaScript for quick page loads

### Design Features
- **Modern UI**: Clean, professional design with gradient accents
- **Responsive Layout**: Fully responsive on desktop, tablet, and mobile devices
- **Smooth Animations**: Fade-in effects and smooth scrolling for better UX
- **Interactive Elements**: Hover effects and transitions on cards and buttons
- **Accessible**: ARIA labels, skip links, and keyboard navigation support

### Sections
1. **Hero Section**: Eye-catching introduction with clear call-to-action
2. **Services**: Detailed grid showcasing all design services
3. **Process**: Step-by-step workflow explanation
4. **About**: Company philosophy and values
5. **Contact**: WhatsApp integration with special offer

## Technology Stack

- **HTML5**: Semantic markup with SEO best practices
- **CSS3**: Custom styles with CSS Grid and Flexbox
- **JavaScript**: Vanilla JS for interactivity and animations
- **Google Fonts**: Inter font family for modern typography

3. For production, upload all files to your web hosting service
4. Update the domain in meta tags if different from htdesignlabs.com

## Customization

### Update Contact Information
- Find and replace `+919952593199` with your WhatsApp number
- Update `htdesignlabs.com` with your actual domain

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --primary-dark: #4f46e5;
    --secondary-color: #10b981;
}
```

### Add Portfolio
When ready, create a new section:
```html
<section id="portfolio" class="portfolio">
    <!-- Add your portfolio content here -->
</section>
```

## Performance Optimization

- Images should be optimized (use WebP format when possible)
- Consider adding a favicon.ico file
- Minify CSS and JS for production
- Enable gzip compression on your server
- Use a CDN for faster global delivery
